# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0

Initial public version.

Main features:

- Python weather script for a custom Waybar module
- JSON output mode for Waybar integration
- TUI mode for a larger terminal weather dashboard
- Hyprland-friendly popup flow for the TUI window
- current conditions, hourly forecast, daily forecast, and alerts
- location helpers for manual geocoding, IP lookup, and GeoClue checks
- cache support for weather data and live location checks
